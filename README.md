# LHR Choked Flow Calculations

This repository contains calculations and utilities for LHR (Longhorn Rocket) choked flow analysis.

## Files

- `main_calcs.py` - Main calculation scripts
- `unit_conversions.py` - Unit conversion utilities using Pint

## Requirements

- Python 3.x
- pint (for unit conversions)

## Installation
